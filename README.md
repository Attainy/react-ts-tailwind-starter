# How to Use
## 1. Clone
### 현재 폴더에 설치
```
git clone https://github.com/Attainy/react-ts-tailwind-starter.git .
```

### 새로운 폴더
```
git clone https://github.com/Attainy/react-ts-tailwind-starter.git 프로젝트이름
```

## 2. 
```
npm install
```

# Install List
- React
- TypeScript
- Vite
- Tailwind CSS (+ 설정)
- Tailwind-Merge
- Tailwind-Form
